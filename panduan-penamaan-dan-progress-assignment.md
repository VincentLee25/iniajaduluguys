## Ketentuan Penamaan

### 1. Judul Community WhatsApp

Format judul community WhatsApp untuk grup program kerja:

```text
[JENIS KEGIATAN] [REGION] - JUDUL KEGIATAN
```

Contoh:

```text
LDK-CP ALS - IMMERISE
```

Jika tidak ada judul kegiatan, maka judul kegiatan tidak perlu ditambahkan.

Contoh tanpa judul kegiatan:

```text
LDK-CP ALS
```

### 2. Grup di Dalam Community WhatsApp

Setiap grup di dalam community WhatsApp harus mengikuti format berikut:

```text
[JENIS KEGIATAN] DIVISI NAMA
```

Contoh:

```text
LDK-CP DIVISI FUNDING
```

Selain grup divisi, wajib ada juga grup berikut:

```text
[JENIS KEGIATAN] ALL DPI
[JENIS KEGIATAN] ALL PANITIA
```

Contoh:

```text
LDK-CP ALL DPI
LDK-CP ALL PANITIA
```

### 3. Meeting Title di Masterlink

Folder `meeting title` yang ada di Masterlink harus mengikuti format dan ketentuan yang sudah ada di sana.

Meeting title di Masterlink wajib memiliki agenda spesifik yang mencakup seluruh pembahasan rapat.

### 4. Folder OneDrive

Format penamaan folder OneDrive:

```text
JENIS KEGIATAN - NAMA KEGIATAN
```

Contoh:

```text
LDK-CP - IMMERISE
```

Jika tidak ada nama kegiatan, cukup gunakan jenis kegiatannya saja.

Contoh:

```text
LDK-CP
```

### 5. Nama Rapat di Notulen dan Folder

Nama rapat di notulen dan folder cukup menggunakan bagian depan dari meeting title di Masterlink.

Jika format folder OneDrive adalah:

```text
YYYY-MM-DD Coordination I
```

maka nama rapat di notulen/folder cukup:

```text
YYYY-MM-DD Coordination I
```

Untuk rapat divisi, kata `Coordination` diganti sesuai nama divisi.

Contoh:

```text
YYYY-MM-DD Funding I
```

Ingat: nama judul rapat tetap harus mengacu pada `meeting title` yang ada di Masterlink.
